#CHEXPLORE

CHEXPLORE is a virtual chemistry lab application that allows users to safely explore and experiment with chemical reactions in a fun, interactive, and visual way. Users can mix chemicals in a virtual container and observe real-time reaction results, such as color changes, gas formation, and precipitation.

Overview

CHEXPLORE provides a safe and interactive platform for learning chemical reactions. Instead of only reading theory, users can actively experiment and observe results, making chemistry more engaging and easier to understand.

##Features
Mix different chemicals in a virtual lab.
Real-time visualization of reactions (color change, gas evolution, precipitation).
Drag-and-drop interface for easy interaction.
Safe experimentation without physical risks.
